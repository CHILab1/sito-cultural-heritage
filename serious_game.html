Serious Game

#flagshipdigitaltechnologies #wp6 #task3

PURPOSE

The proposal includes a serious game in video game format set in the Norman Castle of Aci Castello (CT), offering an interactive experience associated with historical learning. Through immersive missions and storytelling, players discover the castle's history, artifacts, and legends linked to the ancient village of Aci Castello. The game aims to provide educational value by enhancing cultural heritage, making it both engaging and informative.

DEVELOPMENT | STEP 1

A first version of the serious game has been created using a point-and-click game approach, combining photographic images of the castle with 3D-rendered interactive elements. This initial version focuses on exploring various areas within the castle grounds and interacting with objects that provide historical information and challenges that enrich the learning experience in an engaging manner.

DEVELOPMENT | STEP 2

The advanced version of the serious game includes more levels, introduces new challenges, characters, and quests that delve deeper into different historical events. These developments make use of advanced graphics technology to create greater depth and involvement within the game's storyline while maintaining educational goals.
